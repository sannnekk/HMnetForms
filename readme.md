# HMnetForms

Shopware 6 Administration module for managing custom forms (HMnet Forms).

## Features

- Create and manage forms and fields
- Multi-language support with Shopware language switch
- Meteor (mt-\*) component usage in the administration UI

## Development

Build the administration assets after changes:

- `./bin/build-administration.sh`
